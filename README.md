# Project Name - Analyzing Driver Behaviour and Coupons   

## Overview
Goal of this project is to analyze customer behaviour while driving when they receive food coupons in their cell phone, and their coupon acceptance which depends on many factors.      

## Data
Data provided is coupons.csv 

### Requirements 
Analyze the data , plot graphs to see the different customer behaviour, who accepted the coupons or not.

Jupiter Notebook 

###### prompt.ipynb — 

Brief summary:

- **File:** `prompt.ipynb` — interactive Jupyter notebook for exploring coupon data.
- **Purpose:** Load, clean, analyze, and visualize `data/coupons.csv` and related images.
- **Data:** `data/coupons.csv`; image assets in `images/`.
- **Run:** Open with `jupyter notebook prompt.ipynb` or `jupyter lab prompt.ipynb` and run cells.
- **Requirements:** Python 3.8+, `pandas`, `numpy`, `matplotlib`, `seaborn`, and `jupyter`.

Install minimal requirements:

```bash
pip install jupyter pandas numpy matplotlib seaborn
```
